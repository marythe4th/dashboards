# Анализ продаж банковских продуктов

**Цель:** Оптимизация работы колл-центра и выявление узких мест в работе.

**Что сделано:**
* Собрана воронка от звонка до активации продукта.
* Реализована динамика метрик и операционный мониторинг.
* Добавлены фильтры и переключение отображения через параметры.
* Настроена цветовая индикация KPI для быстрого поиска проблемных зон.

**Стек:** Yandex DataLens, SQL.
